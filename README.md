# ICARUS — Aerospace Innovation Platform

Этот проект состоит из двух частей: **Frontend** (React + Vite) и **Backend** (Node.js + Express + Prisma). Для того чтобы проект работал корректно (с авторизацией, базой данных и командами), вам нужно запустить *обе* части одновременно в разных терминалах.

## 🚀 Как запустить проект локально

### 1. Запуск Backend-сервера (База данных и API)
Бэкенд отвечает за сохранение пользователей, проверку паролей (JWT) и работу с базой данных PostgreSQL (через Prisma).

1. Откройте новый терминал.
2. Перейдите в папку сервера:
   ```bash
   cd d:\Icarus_site\server
   ```
3. Запустите сервер:
   ```bash
   node index.js
   ```
*(Сервер запустится на порту `http://localhost:4000`. Не закрывайте этот терминал, пока работаете с приложением!)*

### 2. Запуск Frontend-приложения (Интерфейс)
Фронтенд — это визуальная часть сайта, написанная на React.

1. Откройте **второй** (новый) терминал.
2. Перейдите в корневую папку проекта:
   ```bash
   cd d:\Icarus_site
   ```
3. Запустите локальный сервер Vite:
   ```bash
   npm run dev
   ```
*(Сайт откроется по адресу `http://localhost:5173`. Фронтенд настроен автоматически перенаправлять `/api` запросы на ваш запущенный бэкенд).*

## 🛠️ Команды разработчика (Prisma ORM)

Если вы вносите изменения в структуру базы данных (файл `server/prisma/schema.prisma`), вам необходимо синхронизировать эти изменения с удаленной базой данных Supabase.

Для этого выполните в терминале (находясь в папке `server`):

**Обновить базу данных на основе схемы:**
```bash
npx prisma db push
```

**Сгенерировать клиент (чтобы код "увидел" новые таблицы):**
```bash
npx prisma generate
```

*(Обратите внимание: перед генерацией клиента лучше останавливать Node-сервер `Ctrl+C`, чтобы избежать ошибки блокировки файлов Windows).*

**Полный сброс базы данных (ОСТОРОЖНО):**
```bash
npx prisma migrate reset
```
*(Эта команда **полностью удаляет** все таблицы со всеми данными и пересоздает их пустыми. Используйте, если хотите очистить всех пользователей и команды, чтобы протестировать всё с чистого листа).*
